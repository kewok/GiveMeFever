Percent symptomatic: governs the probability that an infectious host exhibits symptoms

Virulence: governs the probability that an infectious host dies during each time step

R0: the number of secondary infections generated by a single infectious host in an otherwise susceptible population

Incubation period: governs the probability that an exposed host becomes infectious per time step

Recovery rate: governs the probability that an infectious host recovers during each time step

Immunity decay rate: governs the probability that a resistant host loses immunity during each time step
